## Start

### Code Standard

Este projeto segue o code standard do Google para Java/Kotlin.

Baixe/atualize o `code standard` para o projeto e adicione ao seu intellij. Execute o script
abaixo para baixar o arquivo de configuração para o IntelliJ.

```shell
sh scripts/getIntellijGoogleStyle.sh
```

Acesse as configurações do Intellij e import o arquivo de configuração. Na versão `IntelliJ IDEA 
2025.2.4 (Community Edition)`, basta acessar `File -> Settings -> Editor -> Code Style` e
adicionar um novo esquema a partir do arquivo baixado.
